# Monster In Heaven
![Screenshot](http://ludumdare.com/compo/wp-content/compo2/479518/55022-shot0-1440360905.png)
*You're a Monster. In heaven. Eat cakes and kill pixies for bonus points.*

## Controls
| Key       | Description                                                     |
|-----------|-----------------------------------------------------------------|
| A/D       | Move left/right                                                 |
| SPACE     | Kill pixie/eat cake                                             |
